# Recase-Unity
A 3D detective sim (CE capstone) with infinite replayability via GenAI. Players solve procedural cases by gathering evidence and interrogating autonomous NPCs using a free-text NLP interface and a Smart Journal.

Game Link: https://drive.google.com/drive/folders/12m7KxEto6ey5ckgtkjpMksn1E89PUbTO?usp=sharing
